# F.B.S. Launcher 2.0

Simple console launcher that groups daily tasks into Work / Study / Fun.

## Features
- Launch multiple apps / websites with one keypress.
- Simple menu system.
- Pure C++, uses `system()` calls (Windows).

## How to build
- Windows + any C++ compiler (MSVC, MinGW).
- Compile `main.cpp` and run the `.exe`.

## Customization
Edit the file paths inside `switch` statements to match your own software locations.

## Dependencies
- None. Only the C++ standard library.
